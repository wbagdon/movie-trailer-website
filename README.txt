Quick start

- Download package
- Download and install Python 2.7.x (https://www.python.org/)
- Run entertainment_center.py
- HTML and CSS content is automatically generated by fresh_tomatoes.py
- A webpage will be displayed showing the default "Favorite Movies"

Modification

- To add or remove movies, edit the entertainment_center.py "Movies" list
- Content stored for movies can be change in the "Movies" class in media.py
